# Project 01 – Data Understanding

## Objective
The goal of this project is to understand the structure of a dataset and identify basic data characteristics before performing analysis.

## Tasks Performed
- Loaded the dataset
- Checked number of rows and columns
- Identified data types of each column
- Checked for missing values
- Observed basic statistics of the dataset

## Tools Used
- Microsoft Excel
- Python (Pandas)

## Dataset
The dataset used for this project is included in this folder.

## Outcome
This project helped understand how to inspect and prepare raw data before starting deeper analysis.
